# Visitor Logging System

## Project Description
A simple Visitor Logging System built using Hibernate ORM with Java and embedded H2 database. It allows adding visitors and viewing all visitor logs via a simple console interface.

## Prerequisites
- Java 11 or higher
- Maven 3.6+

## Setup Instructions
1. Clone the repository.
2. Navigate to the project directory.
3. Run mvn clean install to build the project.

## Running the Application
Use the following command to run the console application:
   mvn exec:java

## Application Usage
- Choose option 1 to add a visitor.
- Enter visitor details as prompted.
- Choose option 2 to list all visitors.
- Choose option 3 to exit the application.

## Technologies Used
- Java 11
- Hibernate ORM 6
- H2 Embedded Database
- Maven Build Tool
